Projetos Desenvolvidos

PROJETOS DE SIMULADORES

Este projeto objetiva a criação de modelos computacionais (simuladores básicos) de conceitos físicos e estatísticos para serem baixados e utilizados gratuitamente por professores e alunos em sala de aula. Python, A interface é simples e intuitiva, com a funcionalidade de adicionar, editar variáveis, constantes físicas e cenários.

Projeto Explorando o sistema solar

Um jogo de exploração completo onde o jogador poderá explorar de forma lúdica e interativa todo o sistema solar. O jogo foi criado usando as bibliotecas PyGames e Tkinter. Este projeto inclui uma API para integração com um banco de dados MySQL para armazenar informações do usuário e exibir conteúdo dinâmico.

Projetos em Desenvolvimento

PROJETO MATA ATLANTICA, DESMATAMENTO E MICROCLIMA NO CONTEXTO URBANO

Este projeto trata de um estudo direcionado para a compreensão das mudanças climáticas no contexto dos microclimas provocados pelo agravamento do desmatamento do bioma da mata atlântica em um cenário urbano específico da cidade de Conceição de Macabu, localizada no norte do estado do Rio de Janeiro. O projeto está sendo desenvolvido inteiramente utilizandoJupyterLabs.(Para mais informações visite o site do projeto)

PROJETO CLEAN CITY

Este projeto visa mapear pontos de limpeza na cidade e ajudar a população a se mobilizar para realizar a limpeza nos pontos marcados, gerando pontos para os grupos que podem ser acumulados e trocados por descontos nos comércios locais e reconhecimento das autoridades locais. O projeto visa também notificar as autoridades sanitárias da cidade a fim de obter recursos e amparo para o projeto.
